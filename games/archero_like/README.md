# Code Ascension: Mortal Cultivation (问道：凡人修仙传) 🌩️🧘‍♂️

> “凡人一念，逆天改命。穿越异界，以代码解析天道，重走修仙路。”

## 📖 项目简介

本项目是一个 **国风修仙 Roguelike 生存游戏**。

故事讲述了一名现代程序员意外穿越到修仙世界，附身于一个资质平平的废柴凡人身上。为了在残酷的修真界生存，他必须利用前世的“编程思维”来解析功法、优化灵力回路，拜入宗门，从 **练气期** 开始，一步步修炼至 **真仙**，最终渡劫飞升。

无需安装任何依赖，单文件 HTML 即可运行，支持 PC 键盘操作与移动端触屏。

## 🎮 玩法说明

1.  **宗门选择**：
    *   开局需选择拜入的宗门，不同宗门拥有截然不同的核心机制。
    *   ⚔️ **蜀山剑派**（已开放）：主修御剑术，核心机制为【剑心】与【飞剑】，追求极致的单体爆发与多重打击。
    *   📜 *青城玄门*（开发中）：主修符箓与阵法。
    *   🩸 *血煞魔宗*（开发中）：主修献祭与狂暴。

2.  **操作方式**：
    *   **移动**：`WASD` 或 `方向键` (PC) / 屏幕拖拽 (Mobile)。
    *   **攻击**：角色会自动运转功法，锁定最近的敌人攻击。
    *   **暂停/冥想**：`ESC` 键。

3.  **修仙之路**：
    *   **杀妖取丹**：击败妖兽获取灵气（经验）。
    *   **境界突破**：灵气积攒圆满后，通过【机缘三选一】获得新神通或强化属性。
    *   **渡劫飞升**：坚持存活 15 分钟，并击败最终的【九幽魔尊】，即可破碎虚空，成就真仙。
